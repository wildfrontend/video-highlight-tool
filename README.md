# Video Highlight Tool 

## Tech Choices
- Nextjs
- UI Framwork: Mui
- State Management: Zustand

## Local Development

要在本地運行範例，請依照以下步驟操作：

### 安裝依賴：

```jsx
npm install
npm run dev
```

### 訪問應用程式：

打開您的瀏覽器，並前往 http://localhost:3000。
